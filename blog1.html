Introuction to CSS Box Model

CSS , stands for Cascading Style Sheet. It is used to provide styling to web pages, so that our web page is presentable to users. A Web page generally consists of 3 main section – HTML, CSS and javascript. HTML which provides the content on web page , CSS to style the HTML elements and Javascript to to process and store the data from HTML document.

Many developers and newbies use CSS concepts without having a sound knowledge of how they really work, they use the concepts to solve the problem through trial and error method , rather than learning the basics. It is similar to the more you practice the more you learn. But there are some concepts that should be known to the developer for better layout and positioning of HTML Elements. One such concept is CSS Box model.

Today we will learn how to use CSS box model. So Let’s start.

According to CSS Box model , Every element on a web page is represented as a rectangular box. Each box is composed of 4 sections – the content section, the padding section , the border section and the margin section The following image will help you to understand the Box model.



In the above image,
Content – is the actual content of element which can be a image , a text or a video .
Padding – the area around the content. It is transparent.
Border – the area around the content and padding. It can be transparent or visible.
Margin – the area which holds the entire content with padding and margin . It is usually transparent.

Now let us understand each section separately with examples.
